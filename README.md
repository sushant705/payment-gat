# College Project Resource Fee Payment Gateway

A lightweight front-end payment gateway demo for college project submission.

## What it does

- Displays the mandatory **₹8,000 resource fee**.
- Clearly mentions the fee is **refundable after successful project completion**.
- Accepts basic student details and simulates a successful payment request.

## Run locally

Because this is a static project, you can open `index.html` directly in a browser.

Or run a local server:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
